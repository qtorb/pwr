BLOQUE H: PERSISTENCIA VISIBLE — Resumen Visual
════════════════════════════════════════════════════════════════════════════════

OBJETIVO: "PWR deja de sentirse como herramienta de tareas aisladas
           y pasa a sentirse como workspace con memoria útil"

════════════════════════════════════════════════════════════════════════════════
1. PROBLEMA QUE RESUELVE
════════════════════════════════════════════════════════════════════════════════

HOY:
  User ejecuta tarea → ve resultado → ¿dónde quedó? → desconfianza
  User vuelve después → ¿qué hice? → no hay historial visible
  User ve proyecto → solo lista de tareas, sin "historia"

SÍNTOMAS:
  ❌ No siente que nada se guardó
  ❌ No ve continuidad
  ❌ No entiende dónde retomar
  ❌ Project se siente como "lista", no "workspace"

IMPACTO:
  → PWR se siente temporal, no como tool real
  → Usuario no confía en retomar trabajo
  → Adopción baja (cada sesión es "nueva")

3. 5 CAMBIOS VISUALES CONCRETOS
════════════════════════════════════════════════════════════════════════════════

A) RESULTADO: Señal clara de guardado
   HOY:   [Resultado] [Botones]
   →      No hay señal de que se guardó

   NUEVO: [Resultado]
          📌 Guardado automáticamente
          Proyecto: X, Tarea: Y
          [CTAs conectados]
   →      User siente "esto está seguro aquí"

B) HOME: Sección "Hoy" muestra historial del día
   HOY:   Grids genéricos (sin contexto temporal)
   →      User no sabe qué es reciente

   NUEVO: Sección "Hoy" arriba
          ✅ Tarea A (hace 10 min)
          ✅ Tarea B (hace 1 hora)
          [Grid de trabajo en progreso]
   →      User ve "historial del día" inmediatamente

C) PROYECTO: Separa ejecutadas vs pendientes
   HOY:   Lista plana de tareas
   →      No sé qué se hizo

   NUEVO: ✅ Ejecutadas [con resultados]
          📌 Pendientes
   →      Proyecto es "workspace con historia"

D) RESULTADO: Opciones de reutilización claras
   HOY:   [Copiar] [Otra] [Proyecto]
   →      No está claro qué puedo hacer con esto

   NUEVO: ✅ Copiar resultado
          📂 Ver en proyecto
          🔄 Reutilizar como contexto
          🎯 Crear tarea basada en esto
   →      Cada acción conecta con siguiente paso

E) RESULTADO: Continuidad post-ejecución
   HOY:   [Resultado] [Botones desconectados]
   →      "¿Qué hago ahora?"

   NUEVO: [Resultado]
          Siguiente paso:
          - Refinar en proyecto
          - Usar en otra tarea
          - Crear tarea relacionada
   →      Flujo continua naturalmente

4. ZONAS QUE CAMBIAN (Visuales solamente)
════════════════════════════════════════════════════════════════════════════════

Resultado (display_onboarding_result):
  + Sección "Guardado automáticamente"
  + Contexto visible (proyecto, tarea)
  + CTAs reorganizados

Home (home_view):
  + Sección "Hoy" (historial)
  + Badges de estado (✅ ejecutado, 📌 pendiente)
  + Timestamps ("hace X minutos")

Proyecto (project_view):
  + Separación ejecutadas / pendientes
  + Badges de resultado disponible
  + Historial visible

QUÉ NO CAMBIA:
  ❌ Backend (save, execute, BD schema)
  ❌ Router (decisiones eco/racing)
  ❌ Auth/multiusuario
  ❌ Onboarding flujo

6. VALIDACIÓN: ¿PWR se siente como workspace?
════════════════════════════════════════════════════════════════════════════════

Test 1: "¿Dónde quedó tu trabajo?"
  ✅ ÉXITO: User señala resultado → "Aquí, guardado en proyecto X"
  ❌ FALLO: "No sé, desapareció"

Test 2: "¿Qué hiciste hoy?"
  ✅ ÉXITO: User señala Home "Hoy" → "Resume (10 min), Escribe (1 hora)"
  ❌ FALLO: "Tengo que buscar en grids"

Test 3: "¿Puedes reutilizar esto?"
  ✅ ÉXITO: "Puedo copiar, usar contexto, crear basada en esto"
  ❌ FALLO: "Solo copiar el texto"

Test 4: "¿Qué sigue?"
  ✅ ÉXITO: "Puedo refinar, crear tarea relacionada, o ver proyecto"
  ❌ FALLO: "No sé, quedé aquí"

Test 5: "¿Cómo es este proyecto?"
  ✅ ÉXITO: "2 tareas hechas, 1 pendiente. Veo resultados de hechas"
  ❌ FALLO: "Solo lista, sin contexto"

Test 6: "¿Vuelves a un resultado anterior?"
  ✅ ÉXITO: "Lo veo en Home o en Proyecto. Clic y vuelvo"
  ❌ FALLO: "No está, desapareció"

MÉTRICA GLOBAL:
  "¿Cómo describís PWR ahora?"
  ✅ ÉXITO: "Espacio donde mi trabajo queda guardado"
  ❌ FALLO: "Tool que ejecuta tareas que desaparecen"
